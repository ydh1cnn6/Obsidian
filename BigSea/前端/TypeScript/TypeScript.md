# 教程
1、安装
```shell
npm install -g typescript
```

2、编译成JavaScript
```shell
tsc greeter.ts
```

>[!error] 
>如果编译时报错，`greeter.js`文件还是被创建。 就算你的代码里有错误，你仍然可以使用TypeScript。但在这种情况下，TypeScript会警告你代码可能不会按预期执行。
