# 文本块
#解决多行字符串问题 #Java13
```java
// 传统多行字符串（繁琐）
String html = "<html>\n" +
              "  <body>\n" +
              "    <p>Hello</p>\n" +
              "  </body>\n" +
              "</html>";

// 文本块（简洁）
String html = """
              <html>
                <body>
                  <p>Hello</p>
                </body>
              </html>""";
```


# 字符串模板（String Templates）：
支持变量插值 #Java21
```java
String name = "Alice";
int age = 30;

// 字符串模板（使用`STR`处理器）
String info = STR."Name: \{name}, Age: \{age}"; 
// 等价于："Name: Alice, Age: 30"
```
