<font style="color:rgb(77, 77, 77);">在 Groovy 中，</font><font style="color:rgb(243, 59, 69);">{ ... } </font><font style="color:rgb(77, 77, 77);">代表的是闭包。这意味着你不可以像下面这样创建数组字面量：</font>

```java
int[] array = { 1, 2, 3}
```

<font style="color:rgb(77, 77, 77);">正确的创建方法像下面这样：</font>

```java
int[] array = [1,2,3]
```
